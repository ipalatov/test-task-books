Описание предметной области.
Необходимо создать список книг и авторов с возможностью редактирования.
Книги имеют название, жанр, год издания. Книга может иметь несколько авторов.

Задание
Необходимо разработать следующий функционал:
Страница со списком книг, содержащую 10 книг и постраничную навигацию для вывода остальных.
Возможность фильтрации книг по автору, жанру, году издания, сортировка по названию и году издания.

Страница со списком авторов с выводом информации по количеству книг для каждого автора.

Страница добавления, редактирования и удаления книг.
Страница добавления, и редактирования и удаления авторов.

При добавлении и редактировании данные должны проходить валидацию:
- авторы должны быть уникальными;
- книги должны быть уникальными по названию;
- название книги не более 250 символов, год не больше текущего.

Инструменты для разработки
php, javascript (можно использовать Jquery), можно использовать Bootstrap.
